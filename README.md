## Python 2.7 vs Python 3.x
<pre>
1. input() and raw_input()
   *python 2.7: raw_input()- to get input in string format
   *python 3.x: input()- to get input in string format
   
2. division operator
    *python 2.7: 7/2=3	
    *python 3.x: 7/2=3.5            
                 7//2=3   
                 
3. print syntax
    *python 2.7: print “Hello World!”
    *python 3.x: print(“Hello World!”)
    
</pre>
